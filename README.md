# 📦 Packet Debugger - OSI Layers Game

An interactive cybersecurity learning game that teaches the OSI Model through packet delivery and network troubleshooting challenges.

## 🎮 About

Packet Debugger is a browser-based educational game where players:

1. Arrange the OSI layers in the correct order.
2. Select a packet type.
3. Debug network issues across all 7 OSI layers.
4. Restore the network and successfully deliver the packet.

The game provides a hands-on way to understand networking concepts and common troubleshooting scenarios.

## 🚀 Features

- Interactive OSI Model Learning
- Network Troubleshooting Simulation
- Layer-by-Layer Debugging
- Realistic Error Logs
- Multiple Packet Types
- Progress Tracking
- Responsive Design
- Modern Cybersecurity-Themed Interface
- Confetti Success Animation

## 🧠 Concepts Covered

### OSI Layers

- Application Layer
- Presentation Layer
- Session Layer
- Transport Layer
- Network Layer
- Data Link Layer
- Physical Layer

### Networking Topics

- DNS Resolution
- HTTP/HTTPS
- TCP/IP
- Routing
- ARP
- Ethernet
- TLS Encryption
- Session Management
- Signal Transmission

## 🐛 Troubleshooting Scenarios

Players debug issues such as:

- DNS Failures
- Port Mismatches
- TLS Handshake Errors
- Session Timeouts
- Checksum Errors
- Routing Problems
- ARP Failures
- Frame Errors
- Signal Loss
- Cable Failures
